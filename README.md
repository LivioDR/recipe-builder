## Getting Started

First, set up your Google Gemini key on a `.env` file

```
GOOGLE_GENERATIVE_AI_API_KEY=*************************
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
